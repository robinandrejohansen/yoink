Everything in one little app

Built to just work

No browser extensions, no sketchy ad-walls, no "premium" upsell. Open it, paste or search, download.

🎞️

Full quality & fps

Up to 4K, 60fps preserved. Defaults to H.264/AAC so files play in QuickTime, iOS — anywhere — with a max-quality AV1 mode for the purists.

🔎

Search + Shorts filter

Find videos without leaving the app. One click sorts Shorts from long-form, with durations and channels at a glance.

👁️

Hover to preview

Mouse over any result for an instant muted preview. Make sure it's the right video before you download it.

🔒

100% local & private

Runs on your machine. Your searches and files never touch a server — because there is no server. Nothing to log, nothing to leak.

⚡

One file, zero build

A single Flask app wrapping yt-dlp + ffmpeg. No accounts, no telemetry, no Docker. Live progress bar included.

🧡

Free & open source

MIT licensed. Fork it, ship it, make it yours. Built in a weekend and given away — because tools like this should be free.

The honest part

Why local, not a website?

Because a hosted downloader gets IP-blocked by YouTube within minutes and is a legal piñata — that's why every "online downloader" is either broken, buried in ads, or both. Running on your own machine means it actually works, it's private, and it's yours. The constraint is the feature.

Up and running in 30 seconds

Quick start

Needs Python 3 and ffmpeg (brew install ffmpeg on macOS).

Copy 
# clone & enter
git clone https://github.com/robinandrejohansen/yoink && cd yoink

# set up + install
python3 -m venv .venv && ./.venv/bin/pip install -r requirements.txt

# run it
./.venv/bin/python app.py   # → http://127.0.0.1:5001
